Installation

Get started with Windows, OS X, Linux and Android.

Get started with the pre-compiled binary package of the Machinecoin-Wallet for Windows:

1. Unzip the Machinecoin-Wallet to for example "C:\Machinecoin\"
2. Start "machinecoin-qt.exe" with a click on your mouse and close it again after the very first start.
3. Download the example configuration file machinecoin.conf to "C:\Users\<Username>\AppData\Roaming\Machinecoin\machinecoin.conf" and change <rpcusername> and <rpcpassword> inside of it accordingly to your needs.
4. Start "machinecoin-qt.exe" with a click on your mouse

Get started with the pre-compiled binary package of the Machinecoin-Daemon for Windows:

1. Unzip the Machinecoin-Daemon to for example "C:\Machinecoin\"
2. Now open up a terminal and enter the following commands
$cd C:\Machinecoin
$machinecoind
3. After the very first start you'll see some instructions to create a config-file and you'll also get to know where this file should be created. Because the correct path is related to the username of the user that is logged in it would look something like this
"C:\Users\<Username>\AppData\Roaming\Machinecoin\machinecoin.conf"
4. Now open up a terminal and enter the following commands
$cd C:\Users\<Username>\AppData\Roaming\Machinecoin
$notepad machinecoin.conf
After the config-file opens up accept to create that file and enter the following lines into the config-file just right before you hit the save button.
rpcuser=<rpcusername>
rpcpassword=<rpcpassword>
rpcport=40332
server=1
listen=1
daemon=1
gen=0
rpcallowip=127.0.0.1
To earn your first Machinecoins click on the "machinecoind.exe" and open a second terminal to start the mining with 1 processor:
$cd C:\Machinecoin
$machinecoind setgenerate true 1
Other useful commands:
$machinecoind setgenerate false
$machinecoind getinfo
$machinecoind getmininginfo
$machinecoind getbalance
$machinecoind help

Get started with the pre-compiled binary package of the Machinecoin-Wallet for Android:

1. Download the Machinecoin-Wallet for Android to your smartphone or tablet.
2. Open the .apk-file and walk through the installation procedure.
3. Start the app by clicking on the Machinecoin symbol on your homescreen.
4. For further informations have a look at the guide "How to use the Machinecoin Android App right?" at the  Machinecoindoc Repository at GitHub.
